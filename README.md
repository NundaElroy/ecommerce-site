> A comprehensive web app for an electronics ecommerce site with an admin dashboard 

## Install

```sh
git clone https://github.com/NundaElroy/ecommerce-site.git
```

## Usage

```sh
mvn install 
mvn spring-boot:run
```

## Run tests

```sh
mvn test
```
